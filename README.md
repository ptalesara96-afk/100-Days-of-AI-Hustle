## 🤖 AI Dev Tools

### AI Commit Agent
Auto-generates conventional commit messages using GPT-4o-mini.

**Setup (run once):**
```bash
bash scripts/setup_ai_tools.sh
```

**Usage:**
```bash
git add .
git agent-commit
```

**Or via VS Code:**
`Ctrl+Shift+P` → Run Task → `🤖 AI Commit Agent`

**Requirements:**
- Python 3.10+
- `OPENAI_API_KEY` in your `.env` file
